## Demo notebooks for vmango-lab

A demo repository for [vmango-lab](https://github.com/jvail/vmango-lab)

### Setup a conda environment and run locally

```bash
conda env create -f environment.yml
```

```bash
conda activate vmango-lab
jupyter notebook
```
